若控件为垂直布局，子控件默认顺序为从上到下；若控件为水平布局，子控件默认顺序为从左到右。

UI 布局结构如下：
- 根窗口控件：垂直布局
  - 基本信息栏
  - 主区域（占用几乎所有区域）：水平布局
    - 侧边栏
    - 内容展示区：可变区域
  - 状态栏

可变区域类型：

- 总览页：垂直布局
  - 标题
  - 设备信息列表
  - 按钮区：水平布局，默认居右
    - 刷新按钮
    - 管理文件按钮
    - 查看具体信息按钮
    - 安全弹出按钮
- 文件管理页：垂直布局
  - 当前路径显示：水平布局
    - 返回上一层目录按钮
    - 路径显示框
  - 工具栏
    - 创建
    - 复制
    - 剪切
    - 粘贴
    - 删除
    - 重命名
  - 当前目录列表
    - 名称
    - 修改日期
    - 类型
    - 大小

控件层级：

- view
  - 总览页
  - 文件管理页
- widget
  - 剩下所有控件
- dialog
